Sampling Distributions An interactive exploration of $U = h(Y_1, Y_2, \ldots, Y_n)$

We often want the one-dimensional distribution of a function of random variables. Let $Y_1, Y_2, \ldots, Y_n$ be a random sample, and define $$U = h(Y_1, Y_2, \ldots, Y_n).$$ Such a function is called a statistic. We start with $n$ unknowns together with a lot of symmetry, and we reduce the problem to a single random quantity $U$ — whose distribution (the sampling distribution) has properties we can study and exploit.

Below we explore four of the most important sampling distributions through simulation: the distribution of the sample mean, the chi-squared, the Student's $t$, and the $F$ distribution.

1. The Sample Mean: Rolling Dice

Roll a fair six-sided die $n$ times to get $Y_1, Y_2, \ldots, Y_n$, and form the statistic $$U_n \;=\; \bar{Y} \;=\; \frac{Y_1 + Y_2 + \cdots + Y_n}{n}.$$ Each $Y_i$ has $\mu = 3.5$ and $\sigma^2 = 35/12 \approx 2.9167$. So $$E(\bar{Y}) = \mu = 3.5, \qquad \mathrm{Var}(\bar{Y}) = \frac{\sigma^2}{n} = \frac{2.9167}{n}.$$

B. Simulate $U_n$ many times — what is its sampling distribution?

A single value of $U_n$ is just one realization. To see the distribution of $U_n$, repeat the experiment $1000$ times and make a histogram. Try different sample sizes:

Sample size:
Empirical density of $U_n$ (1000 simulations)
Think about it. What do you notice as $n$ grows? Where is the histogram centered? How does its width change? What about its shape?
Discussion

Three things should jump out:

  1. The histogram is always centered near 3.5, regardless of $n$. This matches $E(\bar{Y}) = \mu = 3.5$.
  2. The histogram gets narrower as $n$ grows. The standard deviation is $\sigma/\sqrt{n}$:
    • $n = 4$: SD $\approx 0.854$
    • $n = 25$: SD $\approx 0.342$
    • $n = 100$: SD $\approx 0.171$
    Each fourfold increase in $n$ halves the standard deviation.
  3. The histogram becomes more bell-shaped (more "normal") as $n$ grows — even though the underlying die is uniform, not normal! This is the Central Limit Theorem in action.

Reproduce this in R

R Sampling distribution of the sample mean of n dice 
# --- One realization of U_4 ---
n <- 4
Y <- sample(1:6, size = n, replace = TRUE)
U <- mean(Y)
Y;  U

# --- 1000 realizations: build the sampling distribution ---
trials <- 1000
n      <- 4
U_vals <- replicate(trials, mean(sample(1:6, n, replace = TRUE)))

hist(U_vals, breaks = 25, freq = FALSE, col = "lightblue",
     main = bquote("Sampling distribution of " * bar(Y) * " for n = " * .(n)),
     xlab = expression(bar(Y)))

# Compare empirical to theoretical mean and SD
sigma2 <- 35 / 12          # variance of one fair die
cat("Empirical mean:  ", mean(U_vals), "\n",
    "Theoretical mean:", 3.5,           "\n",
    "Empirical SD:    ", sd(U_vals),    "\n",
    "Theoretical SD:  ", sqrt(sigma2 / n), "\n", sep = "")

# --- Try n = 25 and n = 100 to watch the CLT kick in ---
par(mfrow = c(1, 3))
for (n in c(4, 25, 100)) {
  U_vals <- replicate(1000, mean(sample(1:6, n, replace = TRUE)))
  hist(U_vals, breaks = 30, freq = FALSE, col = "lightblue",
       main = paste("n =", n), xlab = expression(bar(Y)))
}
par(mfrow = c(1, 1))

An important point

The sampling distribution can be defined for any function $U = h(Y_1, \ldots, Y_n)$ — not only the sample mean. The next three sections explore three classical statistics built from a normal sample:

StatisticBuilt fromSampling distribution
$\sum_{i=1}^n Z_i^2$ $Z_i \sim N(0,1)$ $\chi^2_n$
$T = \dfrac{\sqrt{n}(\bar{Y} - \mu)}{S}$ $Y_i \sim N(\mu, \sigma^2)$ $t_{\,n-1}$
$F = \dfrac{S_1^2}{S_2^2}$ two normal samples (equal variance) $F_{\nu_1,\,\nu_2}$

2. The Chi-Squared Distribution

Theorem 7.2. If $Z_1, Z_2, \ldots, Z_n$ are independent standard normal, then $$W = \sum_{i=1}^{n} Z_i^2 \sim \chi^2_n.$$ A $\chi^2_n$ random variable has mean $n$ and variance $2n$.

Let's check this empirically. Take a sample of size $n = 4$ from $N(0,1)$, compute $U = Z_1^2 + Z_2^2 + Z_3^2 + Z_4^2$, and repeat. The histogram should match the $\chi^2_4$ density.

3. The Student's $t$ Distribution

Theorem 7.4. Let $Y_1, \ldots, Y_n$ be a random sample from $N(\mu, \sigma^2)$. Then $$T \;=\; \frac{\sqrt{n}\,(\bar{Y} - \mu)}{S} \;\sim\; t_{\,n-1}.$$

Note we are standardizing using $S$, the sample standard deviation, not the population $\sigma$. That extra source of randomness gives $T$ heavier tails than the standard normal.

We'll take a sample of size $n = 5$ from $N(\mu = 3, \sigma^2 = 16)$, compute $\bar{Y}$ and $S$, then form $$T = \frac{\sqrt{5}\,(\bar{Y} - 3)}{S}.$$ Repeating this gives a histogram that should approach a $t$ distribution with $4$ degrees of freedom. (Note: the population $\sigma^2 = 16$ never appears in $T$ — that's the whole point.)

Compare: The $t_4$ density and the $N(0,1)$ density are both centered at zero and symmetric. How are they different? Look at the tails and at the height near $0$.
Discussion

The $t_4$ density is shorter at the peak and fatter in the tails than the standard normal. Because we estimate $\sigma$ by $S$, occasional small values of $S$ can produce extreme values of $T$ — that's why the tails are heavier. As $n \to \infty$, $S \to \sigma$ and $t_{n-1} \to N(0,1)$.

Reproduce this in R

R T-statistic from a normal sample → t-distribution 
# Population: N(mu = 3, sigma^2 = 16), sample size n = 5
trials <- 1000
n     <- 5
mu    <- 3
sigma <- 4          # since sigma^2 = 16

# Each trial: draw a sample, compute Ybar and S, form T
T_vals <- replicate(trials, {
  Y <- rnorm(n, mean = mu, sd = sigma)
  sqrt(n) * (mean(Y) - mu) / sd(Y)
})

# Histogram with t_4 and N(0,1) overlays
hist(T_vals, breaks = 50, freq = FALSE, xlim = c(-6, 6),
     col = "lightblue", xlab = "T",
     main = expression("Simulated T = " * sqrt(5) *
                       (bar(Y) - 3) / S * " vs " * t[4]))
curve(dt(x, df = n - 1), add = TRUE, col = "firebrick",   lwd = 2)
curve(dnorm(x),          add = TRUE, col = "darkgray",    lwd = 2, lty = 2)
legend("topright",
       legend = c(expression(t[4]), expression(N(0, 1))),
       col = c("firebrick", "darkgray"),
       lwd = 2, lty = c(1, 2), bty = "n")

# Theory: P(|T| > 2.776) = 0.05  for t_4
cat("Empirical P(|T| > 2.776):", mean(abs(T_vals) > 2.776),
    " (theory: 0.05)\n")

4. The $F$ Distribution

Theorem 7.5. If $W_1 \sim \chi^2_{\nu_1}$ and $W_2 \sim \chi^2_{\nu_2}$ are independent, then $$F \;=\; \frac{W_1 / \nu_1}{W_2 / \nu_2} \;\sim\; F_{\nu_1,\,\nu_2}.$$ A useful corollary: if we have two independent samples (sizes $n_1$ and $n_2$) from normal populations with the same variance, then $$\frac{S_1^2}{S_2^2} \;\sim\; F_{n_1 - 1,\, n_2 - 1}.$$

We'll generate two independent samples from $N(0, 1)$: size $n_1 = 6$ and size $n_2 = 10$. Compute $S_1^2$ and $S_2^2$, and form $F = S_1^2 / S_2^2$. The resulting histogram should match $F_{5,9}$.

Empirical density of $F$
$F_{5,9}$ density
Notice: The $F$ distribution is right-skewed and lives on $(0, \infty)$. Because both samples come from populations with the same variance, $F$ tends to cluster near $1$, but with a long right tail. From tables, $P(F_{5,9} \le 3.48) = 0.95$. About what fraction of your simulated $F$ values fell at or below $3.48$?

Reproduce this in R

R Ratio of sample variances → F-distribution 
# Two independent samples from the same N(0, 1).
# (Any common variance would do — sigma^2 cancels in the ratio.)
trials <- 1000
n1 <- 6                 # df1 = 5
n2 <- 10                # df2 = 9

F_vals <- replicate(trials, {
  S1sq <- var(rnorm(n1))
  S2sq <- var(rnorm(n2))
  S1sq / S2sq
})

# Histogram with F(5, 9) density overlay
hist(F_vals, breaks = 80, freq = FALSE, xlim = c(0, 6),
     col = "lightblue", xlab = "F",
     main = expression("Simulated " * S[1]^2 / S[2]^2 * " vs " * F[5*","*9]))
curve(df(x, df1 = n1 - 1, df2 = n2 - 1),
      add = TRUE, col = "firebrick", lwd = 2, n = 400)

# Verify the 95th percentile from tables
cat("Empirical P(F <= 3.48):", mean(F_vals <= 3.48),
    " (theory:", round(pf(3.48, 5, 9), 4), ")\n")
cat("Empirical 95th pct:    ", quantile(F_vals, 0.95),
    " (theory:", round(qf(0.95, 5, 9), 4), ")\n")

R distribution functions (a cheat sheet)

R follows a uniform naming convention for distributions: density, probability (CDF), quantile, and random sample.

Distribution Density CDF Quantile Random sample
Normal $(\mu, \sigma)$ dnorm(x, mean, sd) pnorm qnorm rnorm(n, mean, sd)
Chi-squared $(\nu)$ dchisq(x, df) pchisq qchisq rchisq(n, df)
Student's $t$ $(\nu)$ dt(x, df) pt qt rt(n, df)
$F$ $(\nu_1, \nu_2)$ df(x, df1, df2) pf qf rf(n, df1, df2)

For example, $\chi^2_{0.95,\,9} = 16.919$ comes from qchisq(0.95, df = 9) and $P(F_{5,9} \le 3.48) = 0.95$ from pf(3.48, 5, 9).
