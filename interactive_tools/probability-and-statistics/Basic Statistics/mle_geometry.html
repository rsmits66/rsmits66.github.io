The Geometric Meaning of Maximum Likelihood Estimation

Three views of the same idea: the likelihood is a function of the parameter, and the MLE is where it peaks.

1.50

1. Geometric picture

PDF f(x|c) data xi stem height = f(xi|c)

Stems show f(xi|c) at each data point. The likelihood L(c) is the product of those stem heights.

2. Log-likelihood as a function of c

● current c ● MLE ĉ

The MLE is the c at which this curve peaks — equivalently, where its derivative (the score function) is zero:  d/dc ln L(c) = 0. For a smooth concave log-likelihood, that is exactly the standard calculus condition for a maximum.

0.500

The likelihood for y successes in n independent Bernoulli(p) trials is

L(p) = py(1−p)n−y,    ln L(p) = y ln p + (n−y) ln(1−p)

Setting the derivative to zero gives the MLE in closed form:

d/dp ln L(p) = y/p − (n−y)/(1−p) = 0  ⟹  p̂ = y/n

1. Likelihood L(p)

L(p) is a single bump on [0, 1]. It peaks at p̂ = y/n. The orange line is the tangent at the current p — at the MLE it is exactly horizontal.

2. Log-likelihood ln L(p)

● current p ● MLE p̂ tangent (slope = score)

Working with ln L is easier: the score d/dp ln L(p) = y/p − (n−y)/(1−p) is a clean function of p, and the maximum is the unique zero of that score.

1.00

For two parameters the log-likelihood is a function of the pair (μ, σ²). Geometrically it lives on a surface over the (μ, σ) plane; we visualize it as a heatmap with contour lines. The MLE is the highest point — the point where both partial derivatives vanish:

∂/∂μ ln L = (1/σ²)Σ(xi−μ) = 0  ⟹  μ̂ = x̄
∂/∂σ² ln L = −n/(2σ²) + (1/(2σ⁴))Σ(xi−μ)² = 0  ⟹  σ̂² = (1/n)Σ(xi−x̄)²

1. PDF for current (μ, σ)

As you change μ the bump slides; as you change σ it gets wider or narrower.

2. ln L(μ, σ) over the parameter plane

● current (μ, σ) ✕  MLE (μ̂, σ̂)

Brighter = higher log-likelihood. Both partial derivatives are zero at the bullseye.

For Y₁,…,Yₙ ~ Uniform(0, θ), the joint density is 1/θⁿ provided every yᵢ ≤ θ, and zero otherwise:

L(θ) = (1/θⁿ) · 1{θ ≥ max(y₁,…,yₙ)}

L(θ) is discontinuous: it is exactly zero for θ < max(yᵢ), then jumps up and monotonically decreases for θ ≥ max(yᵢ). So the maximum sits at the boundary of the feasible region — derivative-equals-zero fails here. The MLE is

θ̂ = max(y₁, …, yₙ)

1. Geometric picture

The Uniform(0, θ) density is a flat box of height 1/θ. As θ shrinks the box gets taller — but the moment θ falls below any data point, that point lies outside the support and the joint density collapses to 0.

2. Likelihood L(θ)

● current θ ● MLE θ̂ = max(yᵢ)

Note the cliff at θ̂. The maximum sits right on top of the cliff, not at an interior critical point.

Tag K fish, release them, later catch n fish and observe k of them tagged. Assuming a closed, well-mixed population, the count of tagged fish in the second catch is hypergeometric, so the likelihood for the unknown total population N is

L(N) = C(K, k) · C(N − K, n − k) / C(N, n)

N is a positive integer, so we cannot just set a derivative to zero. Instead, examine the ratio L(N+1)/L(N): it is > 1 while N is below the peak and < 1 once N passes it. Solving L(N+1)/L(N) = 1 gives the closed-form integer MLE

N̂ = ⌊ K · n / k ⌋

With K = 30, n = 50, k = 10 this gives N̂ = ⌊1500/10⌋ = 150.

1. Likelihood L(N)

Each marker is one integer value of N. The likelihood rises, peaks at N̂, and then decays.

2. Log-likelihood ln L(N)

● current N ● MLE N̂

The discrete analogue of "derivative = 0" is "L(N+1)/L(N) crosses 1" — equivalently, ln L stops increasing and starts decreasing.
