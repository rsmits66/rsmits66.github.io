Summation Notation

A concise language for adding things up

Why do we need special notation?

Sometimes mathematical notation helps us express a relationship concisely. In statistics we frequently need to add up a collection of numbers — prices, ages, test scores, frequencies — and it helps to have a compact way to write "add all these values together." This is what summation notation provides.
