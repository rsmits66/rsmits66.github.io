1. One-sample t-test — small-sample mean

Test a population mean against a target value using the t-statistic when n is small. The default values illustrate a small-sample lower-tail test.

Editable data

Classical formula

one-sample t-test, df = n − 1

theoretical p-value
—
from tdf distribution

Simulation R

draw n samples from Normal(μ₀, s²), compute t each time

set.seed(2025)
t_obs <- (ybar - mu0) / (s / sqrt(n))
t_sim <- replicate(100000, {
  d <- rnorm(n, mean = mu0, sd = s)
  (mean(d) - mu0) / (sd(d)/sqrt(n))
})
mean(t_sim <= t_obs)   # for "less"
simulated p-value
—
fraction in tail

The simulation uses s as the population SD when generating null samples, then recomputes sample SDs in each draw. The histogram reproduces the tn−1 shape — slightly heavier tails than the standard normal — so the simulated and theoretical p-values agree whenever the population is exactly normal.

2. Two-sample pooled t-test — small samples, equal variances

Compare two means with small, independent samples assumed to share a common variance. The pooled t-statistic uses df = n₁ + n₂ − 2.

Classical formula

pooled-variance two-sample t, df = n₁ + n₂ − 2

theoretical p-value
—
from tdf distribution

Simulation R

draw both groups from Normal(μ̄, Sp) under H0

set.seed(2025)
mu0  <- (n1*y1 + n2*y2)/(n1+n2)
sp   <- sqrt(((n1-1)*v1+(n2-1)*v2)
            /(n1+n2-2))
t_obs <- (y1-y2-D0)/(sp*sqrt(1/n1+1/n2))
t_sim <- replicate(100000, {
  a <- rnorm(n1, mu0, sp)
  b <- rnorm(n2, mu0, sp)
  spk <- sqrt(((n1-1)*var(a)+
              (n2-1)*var(b))/(n1+n2-2))
  (mean(a)-mean(b))/
    (spk*sqrt(1/n1+1/n2))
})
mean(abs(t_sim) >= abs(t_obs))

3. χ² test for a single variance

Test whether a population variance equals a claimed value. The pivot is (n−1)S²/σ₀² ~ χ²n−1 under H0, with strong dependence on the normality assumption.

Editable data

Classical formula

χ² = (n−1)S²/σ₀², df = n − 1

theoretical p-value
—
from χ²df distribution

Simulation R

draw n from Normal(0, σ₀²), compute (n−1)·var/σ₀²

set.seed(2025)
chi_obs <- (n - 1) * S2 / sig02
chi_sim <- replicate(100000, {
  d <- rnorm(n, mean = 0,
              sd = sqrt(sig02))
  (n - 1) * var(d) / sig02
})
mean(chi_sim >= chi_obs) # upper tail
simulated p-value
—
fraction in tail

Two-sided p-values follow the textbook convention p = 2 · min(P(χ² ≤ χ²_obs), P(χ² ≥ χ²_obs)), capped at 1. The χ² distribution is skewed, so the histogram is decidedly asymmetric — a key visual reminder that not every null distribution is bell-shaped.

4. F test for the ratio σ₁²/σ₂²

Compare two population variances. Numerator df = n₁ − 1, denominator df = n₂ − 1 under H0: σ₁² = σ₂².

Editable data

Classical formula

F = S₁²/S₂², df = (n₁−1, n₂−1)

theoretical p-value
—
from F distribution

Simulation R

draw two normal samples, compute F = S₁²/S₂²

set.seed(2025)
F_obs <- v1 / v2
F_sim <- replicate(100000, {
  a <- rnorm(n1)
  b <- rnorm(n2)
  var(a) / var(b)
})
mean(F_sim >= F_obs)  # upper tail
simulated p-value
—
fraction in tail

Two-sided F p-values use p = 2 · min(P(F ≤ F_obs), P(F ≥ F_obs)). Because the F distribution is asymmetric, the test is not symmetric around 1 — a value like F = 3 and its reciprocal F = 1/3 are not equidistant from the rejection thresholds.
