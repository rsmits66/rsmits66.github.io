1. Medication vs. placebo — two-sample proportion, one-sided

A clinical trial compares a medication to a placebo. H0 is that the two groups share the same underlying success rate.

Editable data
H0: pmed = pplacebo Ha: pmed > pplacebo α: 0.05 (one-sided)

Classical formula

two-proportion z-test, pooled standard error

theoretical p-value
—
P(Z ≥ zobs)

Simulation R

pool under H0, sample both groups from pooled rate

set.seed(2025)
p_null   <- (x1 + x2) / (n1 + n2)
diff_obs <- x1/n1 - x2/n2
diff_sim <- replicate(100000, {
  rbinom(1, n1, p_null)/n1 -
  rbinom(1, n2, p_null)/n2
})
mean(diff_sim >= diff_obs)
simulated p-value
—
fraction with Δp̂ ≥ observed

2. Machine defectives — one-sample proportion, one-sided

A machine must be repaired if it produces more than a target rate of defectives. Test whether the observed rate exceeds the null rate.

Editable data
H0: p = p₀ Ha: p > p₀ α: 0.01 (one-sided)

Classical formula

one-sample z-test, normal approximation to binomial

theoretical p-value
—
P(Z ≥ zobs), normal approx

Simulation R

draw defective counts from Binomial(n, p₀)

set.seed(2025)
Y_sim <- rbinom(100000,
                size = n, prob = p0)
mean(Y_sim >= Y_obs)
simulated p-value
—
fraction of null batches with Y ≥ observed

When np₀ is small the normal approximation becomes shaky — watch the simulated histogram (exact binomial shape) diverge from the symmetric normal bell. The simulated p-value is the one to trust in that regime.

3. Sales contacts — one-sample mean, one-sided

Test whether a sample mean gives evidence that the population mean exceeds a claimed value.

Editable data
H0: μ = μ₀ Ha: μ > μ₀ α: 0.05 (one-sided)

Classical formula

one-sample z-test, large-sample approximation

theoretical p-value
—
P(Z ≥ zobs)

Simulation R

sample from Normal(μ₀, s²), compute means

set.seed(2025)
ybar_sim <- replicate(100000,
  mean(rnorm(n, mean = mu0, sd = s))
)
mean(ybar_sim >= ybar_obs)
simulated p-value
—
fraction of null means ≥ ȳ

When the observed ȳ sits many SEs above μ₀, the simulation may return 0 — a correct reading of "extraordinarily unlikely under H0."

4. Reaction times — two-sample mean, two-sided

Compare two independent-group means with a two-sided test. Evidence against H0 can come from either direction.

Editable data
H0: μ₁ = μ₂ Ha: μ₁ ≠ μ₂ α: 0.05 (two-sided)

Classical formula

two-sample pooled z-test

theoretical p-value
—
2 · P(Z ≥ |zobs|)

Simulation R

pool under H0, use abs() for two-sided

set.seed(2025)
mu0      <- (n1*y1 + n2*y2)/(n1+n2)
sp       <- sqrt(((n1-1)*v1 + (n2-1)*v2)
              /(n1+n2-2))
diff_obs <- y1 - y2
diff_sim <- replicate(100000,
  mean(rnorm(n1, mu0, sp)) -
  mean(rnorm(n2, mu0, sp))
)
mean(abs(diff_sim) >= abs(diff_obs))
simulated p-value
—
fraction with |Δȳ| ≥ |observed|
