How it works

Under H0 (row and column variables independent), expected count for cell (i, j) is Eij = (row i total) × (column j total) / n. The test statistic sums (Oij − Eij)² / Eij across all cells and follows χ²(r−1)(c−1) when n is large. The simulation fixes the row and column totals and shuffles observations under independence — each null-world sample is drawn from a hypergeometric process preserving the marginals, so the simulated distribution matches the "permutation" interpretation of the chi-square test exactly.
