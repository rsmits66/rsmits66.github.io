§ 1   Fundamental Notation

Throughout, vectors are column vectors by convention. We write x' (or x^\top) for the transpose. The following standard matrices arise constantly.

Identity Matrix
\mathbf{I}_n is the n \times n matrix with ones on the diagonal and zeros elsewhere.
Matrix of Ones
\mathbf{J}_n = \mathbf{1}_n\mathbf{1}_n' is the n \times n matrix whose every entry is 1, where \mathbf{1}_n is the n \times 1 vector of ones.
Zero Matrix
\mathbf{0}_{m \times n} is the m \times n matrix of all zeros.

§ 4   The 2×2 Determinant

Definition — Determinant of a 2×2 Matrix
For \mathbf{A} = \begin{bmatrix} a & b \\ c & d \end{bmatrix}, the determinant is \det(\mathbf{A}) = ad - bc.

Geometric meaning: The columns of \mathbf{A} define a parallelogram in \mathbb{R}^2. The signed area of that parallelogram equals \det(\mathbf{A}). A positive determinant means the columns form a right-handed (counter-clockwise) frame; negative means left-handed; zero means the columns are collinear (the matrix is singular).

§ 5   Linear Dependence and Rank

Definition 1.1.12 — Linear Dependence and Rank
Let \mathbf{A} be an n \times s matrix (s \leq n) with column vectors \mathbf{a}_1, \ldots, \mathbf{a}_s. The vectors are linearly dependent if there exist scalars k_1, \ldots, k_s, not all zero, such that k_1\mathbf{a}_1 + \cdots + k_s\mathbf{a}_s = \mathbf{0}. Otherwise, they are linearly independent.

If exactly r \leq s of these vectors are linearly independent and the remaining s - r can be expressed as linear combinations of those r vectors, then \operatorname{rank}(\mathbf{A}) = r.

How to determine rank: Compute the reduced row echelon form (RREF) of \mathbf{A}. Columns that contain a leading 1 (pivot) correspond to linearly independent columns; the remaining columns are dependent (free). The number of pivots equals the rank.

§ 6   Quadratic Forms

Definition 1.1.13 — Quadratic Form
A function f(x_1, \ldots, x_n) = \sum_{i=1}^n\sum_{j=1}^n a_{ij}\,x_i x_j = \mathbf{X}'\mathbf{A}\mathbf{X} is a quadratic form where \mathbf{X} = (x_1,\ldots,x_n)' and \mathbf{A} is an n \times n symmetric matrix with ij-th element a_{ij}.

The symmetric matrix \mathbf{A} is constructed by setting a_{ij} and a_{ji} equal to one-half the coefficient on the x_i x_j term for i \neq j.

§ 7   Orthogonal Matrices

Definition 1.1.14 — Orthogonal Matrix
An n \times n matrix \mathbf{P} is orthogonal if and only if \mathbf{P}^{-1} = \mathbf{P}', so that \mathbf{P}\mathbf{P}' = \mathbf{P}'\mathbf{P} = \mathbf{I}_n.

Writing \mathbf{P} = (\mathbf{p}_1, \mathbf{p}_2, \ldots, \mathbf{p}_n), the necessary and sufficient conditions are:
 (i)   \mathbf{p}_i'\mathbf{p}_i = 1 for each i   (ii)   \mathbf{p}_i'\mathbf{p}_j = 0 for i \neq j.

That is, the columns form an orthonormal set.

§ 9   Positive Semidefinite and Positive Definite Matrices

Definition 1.1.17 — Positive Semidefinite (PSD)
An n \times n matrix \mathbf{A} is positive semidefinite if
 (i)   \mathbf{A} = \mathbf{A}'
 (ii)   \mathbf{Y}'\mathbf{A}\mathbf{Y} \geq 0 for all real \mathbf{Y}
 (iii)   \mathbf{Y}'\mathbf{A}\mathbf{Y} = 0 for at least one nonzero \mathbf{Y}.
Example 1.1.9 — Jn is PSD

\mathbf{J}_n is positive semidefinite: \mathbf{J}_n = \mathbf{J}_n' (symmetric); \mathbf{Y}'\mathbf{J}_n\mathbf{Y} = (\mathbf{1}_n'\mathbf{Y})^2 = \left(\sum y_i\right)^2 \geq 0; and equality holds for \mathbf{Y} = (1, -1, 0, \ldots, 0)'.

Definition 1.1.18 — Positive Definite (PD)
An n \times n matrix \mathbf{A} is positive definite if
 (i)   \mathbf{A} = \mathbf{A}'
 (ii)   \mathbf{Y}'\mathbf{A}\mathbf{Y} > 0 for all nonzero real \mathbf{Y}.
Connection to quadratic forms: If \mathbf{A} is PD, then the quadratic form \mathbf{X}'\mathbf{A}\mathbf{X} is a bowl opening upward — all eigenvalues are positive and the contours are ellipses. If PSD but not PD, the bowl has a flat direction (at least one zero eigenvalue). Try different matrices in the quadratic form visualizer above to see these geometries.

§ 10   Idempotent Matrices

Definition 1.1.19 — Idempotent Matrix
An n \times n matrix \mathbf{A} is idempotent if \mathbf{A}^2 = \mathbf{A}.
It is symmetric idempotent if additionally \mathbf{A} = \mathbf{A}'.

If \mathbf{A} is idempotent of rank n, then \mathbf{A} = \mathbf{I}_n.
