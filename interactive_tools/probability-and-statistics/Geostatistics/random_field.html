How it works

Each field is sampled by circulant embedding with the FFT. The covariance function is tiled onto a torus of size 2N × 2N; its 2D FFT gives the discrete power spectrum S(k); a realization is then Z = Re[ IFFT( √S · W ) ] where W is complex Gaussian white noise. Cropping the toroidal field to N × N suppresses periodicity artifacts. Cost is O(N² log N) per realization.

The spectrum √S is cached per (covariance, ℓ, ν, N). First click rebuilds it; subsequent regenerates only draw fresh white noise and run the IFFT. Watch Setup drop on repeat clicks; changing ℓ, ν, or grid size invalidates the cache.

3D mode displaces the surface vertically by the field value and lights it from above-left. Drag any panel to rotate — all four spin together so the same viewpoint reveals four very different roughness signatures.
