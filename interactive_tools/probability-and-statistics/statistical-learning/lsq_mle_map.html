Three ways to
fit one line.

Least squares, maximum likelihood, and maximum a posteriori — and a prior on the intercept, Gaussian or Laplace, that quietly separates them.

The setup

We fit a straight line y = a·x + b to noisy data, where the noise is Gaussian: yi = a·xi + b + εi,  εi ~ N(0, σ²). All three methods estimate the slope a and intercept b — but only one of them is told anything before seeing the data. Here that belief is a prior placed only on the intercept, symmetric about zero — and you may choose its shape: a Gaussian or a Laplace.

LEAST SQUARES

Minimize the misfit

mina,b  Σi (yi − a·xi − b)2

Pick the line whose vertical residuals have the smallest total square. No probabilities, no assumptions — just geometry.

MAXIMUM LIKELIHOOD

Most probable data

maxa,b  Πi N(yi; a·xi+b, σ²)

With Gaussian noise the log-likelihood is just −RSS⁄(2σ²). Maximizing it is minimizing the sum of squares.

MAXIMUM A POSTERIORI

Belief meets data

mina,b  RSS  +  λ·P(b)

A prior on the intercept adds a penalty P(b), pulling b toward 0. Its shape depends on the prior — b² for a Gaussian, |b| for a Laplace. The slope is left unpenalized.

Because the noise is Gaussian, maximum likelihood and least squares return the identical line — they are the same estimator wearing two names. The genuine contrast in this lecture is between that line and the MAP line.
Choosing the prior

Both priors are symmetric about 0 — both say “the intercept is probably small.” But their shapes differ, and so do the penalties they hand to the MAP objective. This is the ridge-versus-lasso contrast, now acting on a single coefficient. Pick one with the toggle in the experiment below.

GAUSSIAN PRIOR in use

Smooth shrinkage — the ridge penalty

g(b) ∝ exp( −b2 ⁄ 2σb2 )  ⟹  penalty  λ·b2,   λ = σ²⁄σb2

A round, smooth bowl. The MAP intercept is pulled proportionally toward 0 — shrunk, but never landing exactly on it. The ridge-style penalty.

LAPLACE PRIOR in use

Soft-thresholding — the lasso penalty

g(b) ∝ exp( −|b| ⁄ c )  ⟹  penalty  λ·|b|,   λ = 2σ²⁄c

A sharp peak at 0. The MAP intercept is soft-thresholded:  b̂ = sign(b̂LSQ)·max( |b̂LSQ| − λ⁄2A,  0 ). Weak enough data ⟹ b is set exactly to 0. The lasso-style penalty.

The prior on the intercept

What to notice
i.

MLE is least squares in disguise

Gaussian noise makes the log-likelihood a negative multiple of the residual sum of squares. The two objectives have the same minimizer, so the orange MLE line lies exactly under the crimson one.

ii.

The prior is a single penalty term

A prior on b adds exactly one term to the objective: λ·b² for a Gaussian, λ·|b| for a Laplace. Only the intercept is shrunk — the slope keeps a flat prior and moves freely.

iii.

Gaussian shrinks; Laplace selects

Toggle the prior. The Gaussian bowl shrinks b smoothly and proportionally — never to exactly 0. The Laplace peak soft-thresholds it and can set the intercept exactly to 0, pinning the line through the origin. This is the ridge-versus-lasso contrast.

iv.

It helps when the belief is true

When the true intercept sits near 0, either prior trims error from a noisy fit. Push true intercept far from 0 and the same prior becomes a bias. Widen prior strength until λ→0 and every method collapses back onto least squares.
