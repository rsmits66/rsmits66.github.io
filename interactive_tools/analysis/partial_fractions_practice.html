The Cover-Up Toolkit

I. Distinct linear factor
In $\dfrac{P(s)}{(s-a)\,R(s)} = \dfrac{A}{s-a} + \cdots$,   cover the factor $(s-a)$ and evaluate the rest at $s = a$: $$A = \frac{P(a)}{R(a)}.$$
II. Repeated factor — derivative trick
For $(s-a)^n$, let $f(s) = (s-a)^n \cdot \dfrac{P(s)}{Q(s)} = \dfrac{P(s)}{R(s)}$. Then the coefficient of $\dfrac{1}{(s-a)^{n-k}}$ is $$A_{n-k} = \frac{f^{(k)}(a)}{k!}.$$ Top power uses ordinary cover-up; lower powers each cost one derivative.
III. Irreducible quadratic — complex substitution
For $s^2 + bs + c$ with complex root $\alpha$, multiply through by the factor and set $s = \alpha$: $$M\alpha + N = \frac{P(\alpha)}{R(\alpha)}.$$ One complex equation splits into two real ones — match real and imaginary parts to recover $M$ and $N$.
IV. Improper fraction — divide first
If $\deg P \ge \deg Q$, do polynomial long division before anything else: $$\frac{P(s)}{Q(s)} \;=\; \text{quotient}(s) \;+\; \frac{\text{remainder}(s)}{Q(s)}.$$ The remainder is now proper — apply I–III to it.

№ 2 Distinct Linear · Leading Coefficients Pure cover-up
$\displaystyle F(s) \;=\; \frac{5-2s}{6s^2+s-2}$
Factor the Denominator
Quadratic formula on $6s^2+s-2$: roots $s = \tfrac{1}{2}$ and $s = -\tfrac{2}{3}$, giving $(2s-1)(3s+2)$.
$\displaystyle \frac{5-2s}{(2s-1)(3s+2)} \;=\; \frac{A}{2s-1} + \frac{B}{3s+2}$
Find $A$ — cover $(2s-1)$, set $s = \tfrac{1}{2}$
$\displaystyle A \;=\; \frac{5 - 2(\tfrac{1}{2})}{3(\tfrac{1}{2})+2} \;=\; \frac{4}{\tfrac{7}{2}} \;=\; \frac{8}{7}$
Find $B$ — cover $(3s+2)$, set $s = -\tfrac{2}{3}$
$\displaystyle B \;=\; \frac{5 - 2(-\tfrac{2}{3})}{2(-\tfrac{2}{3})-1} \;=\; \frac{\tfrac{19}{3}}{-\tfrac{7}{3}} \;=\; -\frac{19}{7}$
Decomposition
$\displaystyle \frac{5-2s}{6s^2+s-2} \;=\; \frac{8}{7(2s-1)} \;-\; \frac{19}{7(3s+2)}$
№ 3 Distinct Linear · 3 Factors Pure cover-up
$\displaystyle F(s) \;=\; \frac{s^2+5s}{(s+3)(s-1)(s-2)}$
Set Up the Form
$\displaystyle \frac{s^2+5s}{(s+3)(s-1)(s-2)} \;=\; \frac{A}{s+3} + \frac{B}{s-1} + \frac{C}{s-2}$
Find $A$ — cover $(s+3)$, set $s = -3$
$\displaystyle A \;=\; \frac{(-3)^2+5(-3)}{(-3-1)(-3-2)} \;=\; \frac{-6}{20} \;=\; -\frac{3}{10}$
Find $B$ — cover $(s-1)$, set $s = 1$
$\displaystyle B \;=\; \frac{1+5}{(1+3)(1-2)} \;=\; \frac{6}{-4} \;=\; -\frac{3}{2}$
Find $C$ — cover $(s-2)$, set $s = 2$
$\displaystyle C \;=\; \frac{4+10}{(2+3)(2-1)} \;=\; \frac{14}{5}$
Decomposition
$\displaystyle \frac{s^2+5s}{(s+3)(s-1)(s-2)} \;=\; -\frac{3}{10(s+3)} \;-\; \frac{3}{2(s-1)} \;+\; \frac{14}{5(s-2)}$
№ 4 Distinct Linear · Cubic Denominator Pure cover-up
$\displaystyle F(s) \;=\; \frac{6}{2s^3+5s^2+3s}$
Factor the Denominator
Pull out $s$, then factor the quadratic:
$2s^3+5s^2+3s = s(2s^2+5s+3) = s(2s+3)(s+1)$
$\displaystyle \frac{6}{s(2s+3)(s+1)} \;=\; \frac{A}{s} + \frac{B}{2s+3} + \frac{C}{s+1}$
Find $A$ — cover $s$, set $s = 0$
$\displaystyle A \;=\; \frac{6}{(3)(1)} \;=\; 2$
Find $B$ — cover $(2s+3)$, set $s = -\tfrac{3}{2}$
$\displaystyle B \;=\; \frac{6}{(-\tfrac{3}{2})(-\tfrac{3}{2}+1)} \;=\; \frac{6}{(-\tfrac{3}{2})(-\tfrac{1}{2})} \;=\; \frac{6}{\tfrac{3}{4}} \;=\; 8$
Find $C$ — cover $(s+1)$, set $s = -1$
$\displaystyle C \;=\; \frac{6}{(-1)(2(-1)+3)} \;=\; \frac{6}{-1} \;=\; -6$
Decomposition
$\displaystyle \frac{6}{2s^3+5s^2+3s} \;=\; \frac{2}{s} \;+\; \frac{8}{2s+3} \;-\; \frac{6}{s+1}$

№ 6 Repeated at Origin Cover-up + one derivative
$\displaystyle F(s) \;=\; \frac{3s-7}{s^3-4s^2}$
Factor the Denominator
$s^3 - 4s^2 = s^2(s-4)$
$\displaystyle \frac{3s-7}{s^2(s-4)} \;=\; \frac{A}{s^2} + \frac{B}{s} + \frac{C}{s-4}$
Find $A$ — cover $s^2$, set $s = 0$
$\displaystyle A \;=\; \frac{0-7}{0-4} \;=\; \frac{7}{4}$
Find $C$ — cover $(s-4)$, set $s = 4$
$\displaystyle C \;=\; \frac{12-7}{4^2} \;=\; \frac{5}{16}$
Find $B$ — derivative trick
Let $\displaystyle f(s) = s^2 F(s) = \frac{3s-7}{s-4}$.   Then $B = f'(0)$.
$\displaystyle f'(s) \;=\; \frac{3(s-4) - (3s-7)(1)}{(s-4)^2} \;=\; \frac{-5}{(s-4)^2}$
$\displaystyle B \;=\; f'(0) \;=\; \frac{-5}{16} \;=\; -\frac{5}{16}$
Decomposition
$\displaystyle \frac{3s-7}{s^3-4s^2} \;=\; \frac{7}{4s^2} \;-\; \frac{5}{16s} \;+\; \frac{5}{16(s-4)}$

№ 8 Two Irreducible Quadratics Complex substitution × 2
$\displaystyle F(s) \;=\; \frac{2 + 5s + 3s^2 - 4s^3}{(s^2+1)(s^2+4)}$
Set Up the Form
$\displaystyle F(s) \;=\; \frac{Ms+N}{s^2+1} + \frac{Ps+Q}{s^2+4}$
No linear factors here — both quadratics are irreducible. We do complex substitution twice: once at $s=i$, once at $s=2i$.
First quadratic — cover $(s^2+1)$, set $s = i$
$\displaystyle Mi + N \;=\; \frac{2 + 5i + 3i^2 - 4i^3}{i^2 + 4} \;=\; \frac{2 + 5i - 3 + 4i}{3} \;=\; \frac{-1 + 9i}{3}$
(used $i^2 = -1$ and $i^3 = -i$)
Real: $N = -\dfrac{1}{3}$.   Imag: $M = 3$.
Second quadratic — cover $(s^2+4)$, set $s = 2i$
$\displaystyle 2Pi + Q \;=\; \frac{2 + 5(2i) + 3(2i)^2 - 4(2i)^3}{(2i)^2 + 1} \;=\; \frac{2 + 10i - 12 + 32i}{-3}$
$\displaystyle = \frac{-10 + 42i}{-3} \;=\; \frac{10}{3} - 14i$
Real: $Q = \dfrac{10}{3}$.   Imag: $2P = -14 \;\Longrightarrow\; P = -7$.
Decomposition
$\displaystyle \frac{2 + 5s + 3s^2 - 4s^3}{(s^2+1)(s^2+4)} \;=\; \frac{9s - 1}{3(s^2+1)} \;+\; \frac{10 - 21s}{3(s^2+4)}$
№ 9 Improper Fraction Long division + cover-up
$\displaystyle F(s) \;=\; \frac{4s^2 + 5s}{(s-1)(s+3)}$
Check the Degrees
Numerator and denominator are both degree $2$ — improper. Polynomial-divide first.
Polynomial Long Division
$(s-1)(s+3) = s^2 + 2s - 3$.   Divide $4s^2+5s$ by $s^2+2s-3$:
$4s^2 + 5s \;=\; 4(s^2+2s-3) + (-3s + 12)$
$\displaystyle \frac{4s^2+5s}{(s-1)(s+3)} \;=\; 4 + \frac{12 - 3s}{(s-1)(s+3)}$
Decompose the Remainder
$\displaystyle \frac{12 - 3s}{(s-1)(s+3)} \;=\; \frac{A}{s-1} + \frac{B}{s+3}$
$A$ — cover $(s-1)$, set $s = 1$:   $A = \dfrac{12-3}{1+3} = \dfrac{9}{4}$.
$B$ — cover $(s+3)$, set $s = -3$:   $B = \dfrac{12+9}{-3-1} = -\dfrac{21}{4}$.
Decomposition
$\displaystyle \frac{4s^2 + 5s}{(s-1)(s+3)} \;=\; 4 \;+\; \frac{9}{4(s-1)} \;-\; \frac{21}{4(s+3)}$
№ 10 Improper · with Quadratic Long division + complex substitution
$\displaystyle F(s) \;=\; \frac{s^4 + 3}{s^3 + 4s}$
Check the Degrees
Numerator degree $4$, denominator degree $3$ — improper. Divide first.
Polynomial Long Division
Denominator: $s^3 + 4s = s(s^2+4)$.   Divide $s^4 + 3$ by $s^3 + 4s$:
$s^4 + 3 \;=\; s(s^3 + 4s) + (-4s^2 + 3)$
$\displaystyle \frac{s^4 + 3}{s^3 + 4s} \;=\; s + \frac{3 - 4s^2}{s(s^2+4)}$
Decompose the Remainder
$\displaystyle \frac{3 - 4s^2}{s(s^2+4)} \;=\; \frac{A}{s} + \frac{Ms+N}{s^2+4}$
Find $A$ — cover $s$, set $s = 0$
$\displaystyle A \;=\; \frac{3 - 0}{0 + 4} \;=\; \frac{3}{4}$
Find $M, N$ — cover $(s^2+4)$, set $s = 2i$
$\displaystyle M(2i) + N \;=\; \frac{3 - 4(2i)^2}{2i} \;=\; \frac{3 + 16}{2i} \;=\; \frac{19}{2i}$
Rationalize: $\displaystyle \frac{19}{2i} \cdot \frac{-i}{-i} \;=\; \frac{-19i}{2}$
$\displaystyle 2Mi + N \;=\; 0 - \frac{19}{2}i$
Real: $N = 0$.   Imag: $2M = -\dfrac{19}{2} \;\Longrightarrow\; M = -\dfrac{19}{4}$.
Decomposition
$\displaystyle \frac{s^4 + 3}{s^3 + 4s} \;=\; s \;+\; \frac{3}{4s} \;-\; \frac{19s}{4(s^2 + 4)}$

Once you can decompose at sight, Laplace inversion becomes table-lookup. Every numerator and denominator above is a form you'll meet going from $F(s)$ back to $f(t)$.
