A Project in Set Theory

On the Equivalence of (0,1) and [0,1]

A guided construction by way of a clever rearrangement

The purpose of this project is to show that the open interval $(0,1)$ is equivalent to the closed interval $[0,1]$. In the process we will discover that both intervals are equivalent to $[0,1)$ and $(0,1]$. It is then easy to generalize to any interval $[a, b]$ with $a

At first glance the claim is strange: $[0,1]$ contains two points that $(0,1)$ does not — namely the endpoints $0$ and $1$ — and yet we claim the two sets have the same cardinality. The argument exploits the fact that inside $(0,1)$ there is already an infinite sequence we can "slide over" to make room for the missing endpoints.

The five steps
  1. $f$ is one-to-one from $(0,1)$ into $(0,1]$
  2. $f$ maps $(0,1)$ onto $(0,1]$
  3. A bijection $[0,1) \to [0,1]$
  4. $[0,1) \sim (0,1]$
  5. $(0,1) \sim [0,1]$

PreliminariesThe function $f$

Define $f:(0,1) \to \mathbb{R}$ as follows.

For $n \in \mathbb{N},\ n \geq 2$,$\quad f\!\left(\dfrac{1}{n}\right) = \dfrac{1}{n-1}$

and for every other $x \in (0,1)$,$\quad f(x) = x$.

In plain language: $f$ acts as the identity function on almost all of $(0,1)$, but on the special sequence $\tfrac{1}{2}, \tfrac{1}{3}, \tfrac{1}{4}, \ldots$ it shifts each term up to the previous one in the list. The first few values are:

f(1/2) = 1,     f(1/3) = 1/2,     f(1/4) = 1/3,     f(1/5) = 1/4,     f(1/6) = 1/5,     …

The sequence $\tfrac{1}{2}, \tfrac{1}{3}, \tfrac{1}{4}, \ldots$ has no first term missing and no last term — it is infinite in both directions of "density." Sliding every term up by one position opens up a single vacancy at the top (which is filled by the value $1$), while leaving every other point in $(0,1)$ untouched. That is the whole idea.

The five problemsProofs to complete

Each problem below is followed by a hint and a full solution. Work on each part yourself first; reveal the hint only when stuck, and the solution only to check your own work.

Problem 1

Prove that $f$ is a one-to-one function from $(0,1)$ into $(0,1]$.

You must verify two things: first, that every value $f(x)$ lies in the set $(0,1]$, and second, that $f$ sends distinct inputs to distinct outputs.

Hint

Start by checking the range. Split into the two cases from the definition: points of the form $x = 1/n$ with $n \geq 2$, and all other points of $(0,1)$. Where does each type land?

For one-to-oneness, assume $f(x_1) = f(x_2)$ and consider three cases: both inputs are of the form $1/n$; neither is; exactly one is. The third case is the only tricky one — here you'll need the observation that $1/(n-1)$ is itself always of the form $1/m$ with $m \geq 1$.

Solution

Step 1: The image lies in $(0,1]$. Let $x \in (0,1)$.

If $x = 1/n$ for some integer $n \geq 2$, then $f(x) = 1/(n-1)$. Because $n \geq 2$ we have $n - 1 \geq 1$, and therefore $0 < 1/(n-1) \leq 1$. So $f(x) \in (0,1]$.

Otherwise $f(x) = x$, which lies in $(0,1) \subseteq (0,1]$. Either way, $f(x) \in (0,1]$.

Step 2: $f$ is one-to-one. Suppose $f(x_1) = f(x_2)$.

Case A. Both $x_1$ and $x_2$ have the form $1/n$ with $n \geq 2$. Write $x_1 = 1/n_1$ and $x_2 = 1/n_2$. Then $1/(n_1 - 1) = 1/(n_2 - 1)$, which forces $n_1 = n_2$, and hence $x_1 = x_2$.

Case B. Neither $x_1$ nor $x_2$ has that form. Then $f(x_1) = x_1$ and $f(x_2) = x_2$, so $x_1 = x_2$ immediately.

Case C. Exactly one of them has the form. Say $x_1 = 1/n$ for some $n \geq 2$ and $x_2$ is not of that form. Then $f(x_1) = 1/(n-1)$ and $f(x_2) = x_2$. Equality would give $x_2 = 1/(n-1)$. For $n = 2$ this gives $x_2 = 1$, which is impossible since $x_2 \in (0,1)$. For $n \geq 3$ we have $n - 1 \geq 2$, so $x_2$ would be of the form $1/m$ with $m \geq 2$ — contradicting our assumption. Hence this case is impossible.

In every case $x_1 = x_2$, so $f$ is one-to-one. Q.E.D.

Problem 2

Prove that $f$ is a function from $(0,1)$ onto $(0,1]$.

Given any $y \in (0,1]$, you must produce an $x \in (0,1)$ with $f(x) = y$.

Hint

Split on whether $y$ has the form $1/m$ for some positive integer $m$. The set of reciprocals $\{1, \tfrac{1}{2}, \tfrac{1}{3}, \tfrac{1}{4}, \ldots\}$ is exactly the image of the sequence part of $f$. Every other element of $(0,1]$ is fixed by $f$.

Solution

Let $y \in (0,1]$.

Case A. $y = 1/m$ for some integer $m \geq 1$. Let $n = m + 1$, so $n \geq 2$, and let $x = 1/n \in (0,1)$. Then by the definition of $f$, $$f(x) = f\!\left(\tfrac{1}{n}\right) = \tfrac{1}{n-1} = \tfrac{1}{m} = y.$$

Case B. $y$ is not of the form $1/m$. In particular $y \neq 1$, so $y \in (0,1)$; and $y$ is not of the form $1/n$ for any $n \geq 2$. Let $x = y$. Then $x \in (0,1)$ is not of the form $1/n$, so $f(x) = x = y$.

In both cases we have exhibited $x \in (0,1)$ with $f(x) = y$, so $f$ maps $(0,1)$ onto $(0,1]$. Q.E.D.

Combined with Problem 1, we have established that $f$ is a bijection $(0,1) \to (0,1]$. Therefore $(0,1) \sim (0,1]$.

Problem 3

Find a one-to-one function from $[0,1)$ onto $[0,1]$.

You might use the function from Problem 2 with a small modification. The domain now contains the point $0$, and the codomain contains both $0$ and $1$.

Hint

The function $f$ from Problem 2 already hits every point of $(0,1]$ bijectively. All that's missing is the value $0$ — which, conveniently, is already in the new domain $[0,1)$. Just send $0$ to itself.

Solution

Define $g:[0,1) \to [0,1]$ by $$g(x) = \begin{cases} 0 & \text{if } x = 0, \\ \dfrac{1}{n-1} & \text{if } x = \dfrac{1}{n},\ n \in \mathbb{N},\ n \geq 2, \\ x & \text{for every other } x \in (0,1). \end{cases}$$ That is, $g$ extends $f$ by also defining $g(0) = 0$.

$g$ is one-to-one. If $x_1, x_2 \in (0,1)$, the argument from Problem 1 applies verbatim. If $x_1 = 0$ and $x_2 \in (0,1)$, then $g(x_1) = 0$ while $g(x_2) \in (0,1]$ (by Problem 1, step 1), so $g(x_1) \neq g(x_2)$. Thus $g$ is injective on all of $[0,1)$.

$g$ is onto $[0,1]$. By Problem 2, $g$ restricted to $(0,1)$ already covers $(0,1]$. Adjoining $g(0) = 0$, the image becomes $\{0\} \cup (0,1] = [0,1]$.

Therefore $g$ is a bijection $[0,1) \to [0,1]$, and $[0,1) \sim [0,1]$. Q.E.D.

Problem 4

Prove that $[0,1)$ is equivalent to $(0,1]$.

You need a bijection between the two intervals. This one needs no fancy rearrangement.

Hint

Reflecting the interval through its midpoint $1/2$ swaps its endpoints. Try a linear function of the form $h(x) = 1 - x$.

Solution

Define $h:[0,1) \to (0,1]$ by $h(x) = 1 - x$.

Well-defined. If $0 \leq x < 1$, then $0 < 1 - x \leq 1$, so $h(x) \in (0,1]$.

One-to-one. If $h(x_1) = h(x_2)$, then $1 - x_1 = 1 - x_2$, so $x_1 = x_2$.

Onto. Given $y \in (0,1]$, let $x = 1 - y$. Then $0 \leq x < 1$, so $x \in [0,1)$, and $h(x) = 1 - (1 - y) = y$.

Therefore $h$ is a bijection, and $[0,1) \sim (0,1]$. Q.E.D.

Problem 5

Prove that $(0,1)$ is equivalent to $[0,1]$.

All the pieces are now in place. Combine them.

Hint

You have bijections between several pairs of intervals. Compose them. Recall that the composition of bijections is a bijection, and that equivalence of sets ($\sim$) is a transitive relation.

Can you build a chain $(0,1) \to \ (?\!) \ \to \ (?\!) \ \to [0,1]$ using functions you've already defined?

Solution

From the previous problems we have the following bijections:

$\bullet\ $ $f : (0,1) \xrightarrow{\ \sim\ } (0,1]$  (Problems 1 & 2)
$\bullet\ $ $h^{-1} : (0,1] \xrightarrow{\ \sim\ } [0,1),\ \ y \mapsto 1-y$  (Problem 4)
$\bullet\ $ $g : [0,1) \xrightarrow{\ \sim\ } [0,1]$  (Problem 3)

The composition $$F \;=\; g \circ h^{-1} \circ f \;:\; (0,1) \longrightarrow [0,1]$$ is a bijection, being a composition of three bijections. Therefore $(0,1) \sim [0,1]$. Q.E.D.

Equivalently, since equivalence of sets is transitive, the chain $$(0,1) \sim (0,1] \sim [0,1) \sim [0,1]$$ collapses to $(0,1) \sim [0,1]$.

EpilogueGeneralizing to $[a,b]$

For any real numbers $a < b$, the linear map $\ell : [0,1] \to [a,b]$ defined by $\ell(t) = a + (b-a)\,t$ is a bijection, and the same formula (with its domain and codomain restricted) gives bijections between the corresponding open, half-open, and closed intervals. Composing $\ell$ with the bijections constructed above shows that all of $$(a,b), \quad [a,b), \quad (a,b], \quad [a,b]$$ are equivalent. The cardinality of an interval of positive length therefore does not depend on whether its endpoints are included.
