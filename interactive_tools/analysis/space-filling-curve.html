§1 — Introduction

A space-filling curve is a continuous surjection from a one-dimensional interval onto a multi-dimensional region — such as the entire unit square. This challenges intuition: how can a continuous path, which we think of as "one-dimensional," cover every point of a two-dimensional area?

Goal. Construct a continuous function that is surjective — i.e., every point is hit by some .

The construction uses lacunary series: sums whose frequencies grow geometrically (here as powers of 3). The key insight is that the discontinuities of the base function are placed exactly on a Cantor set, so that the corresponding lacunary series is nevertheless continuous.

§5 — The Cantor Set and Surjectivity

The Cantor set consists of numbers in [0,1] whose base-3 expansion uses only the digits 0 and 2. It is a closed, nowhere-dense set of Lebesgue measure zero — yet it has the same cardinality as [0,1].

Key fact. The map Φ restricted to the Cantor set is already surjective onto [0,1]². The construction with interleaved bits gives a point in , and Φ maps C onto the entire unit square. On the middle-thirds gaps, Φ is extended linearly — this is why f was chosen to be linear on (⅓, ⅔).

This is the precise analogue of the Cantor function ("devil's staircase"): a function that rises entirely on the Cantor set but is constant on every removed interval. Here, Φ fills the entire square from the Cantor set alone.

§6 — Properties of Φ

Continuous. Φ(t) is a uniform limit of continuous functions. Both x(t) and y(t) are continuous.
Surjective. Every (x₀, y₀) ∈ [0,1]² is the image of some t₀ constructed by bit-interleaving.
Not injective. Many values of t map to the same point. The curve crosses itself uncountably many times.
Nowhere differentiable (in direction). The curve has no consistent tangent — it is a fractal path at every scale.
