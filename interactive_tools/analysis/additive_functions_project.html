Additive Functions
and their Limits

A Project in Real Analysis

Under what condition on an additive function $f:\R\to\R$ does the limit $\displaystyle\lim_{t\to x} f(t)$ exist at every point $x \in \R$?

in the inquiry-based tradition

To the student

The four problems below are a single, self-contained investigation. Read them in order. Each problem rests on the one before it, and the last sentence of the project is a theorem you will have proved with your own hands.

You are expected to write out a complete proof of every numbered problem. Lemmas you discover along the way — in particular, that $f(0)=0$ and that $f(-x)=-f(x)$ — should be stated and proved separately. The little hint boxes are nudges, not solutions; resist the urge to read them until you have wrestled with the statement on your own.

1.Setup

Definition A function $f:\R\to\R$ is additive if $$f(x+y) = f(x) + f(y) \qquad \text{for every } x,y \in \R.$$

The equation defining additivity is called Cauchy's functional equation. The simplest examples are the linear functions $f(x)=cx$, where $c$ is a fixed real number. Cauchy himself proved in 1821 that these are the only continuous solutions; what is striking, and what this project will help you understand, is that additivity alone is not enough to guarantee continuity. Without some additional regularity, an additive function may behave so erratically that its graph is dense in the plane.

Throughout the project, $f:\R\to\R$ is assumed to be additive.

2.Integer scaling

Problem 1 For each positive integer $n$ and each real number $x$, $$f(nx) = nf(x).$$
Hint Induct on $n$. For the base case $n=1$ there is nothing to do. For the step from $k$ to $k+1$, write $(k+1)x = kx + x$ and apply additivity once.
Two short companion facts are worth recording before you continue, because they are used silently in every problem that follows.
  • Setting $x=y=0$ in the defining equation shows $f(0)=2f(0)$, hence $f(0)=0$.
  • Setting $y=-x$ then gives $0=f(0)=f(x)+f(-x)$, so $f(-x)=-f(x)$.
  • In particular, $f(x-y) = f(x) + f(-y) = f(x) - f(y)$.

3.From local boundedness to small oscillation

For the rest of the project, we make the following standing hypothesis. We will refer to it as (LB), for local boundedness.

(LB) There exist $M>0$ and $a>0$ such that $$|f(x)| \le M \qquad \text{whenever } x \in [-a, a].$$

Problem 2 Assume (LB). Let $\eps > 0$ and choose a positive integer $N$ with $M/N < \eps$. Then $$|x-y| < \frac{a}{N} \;\Longrightarrow\; |f(x) - f(y)| < \eps.$$
Hint Run the following chain in your head before writing it out.
  • (a) If $|x-y| < a/N$, then $N(x-y)$ lies in $[-a,a]$. Why?
  • (b) Hence $|f(N(x-y))| \le M$ by (LB).
  • (c) By Problem 1, $f(N(x-y)) = N f(x-y)$.
  • (d) Combine: $|f(x-y)| \le M/N < \eps$.
  • (e) Finally use $f(x-y) = f(x) - f(y)$ from the remark above.
Notice what Problem 2 actually says: a single $N$ controls the oscillation of $f$ everywhere on $\R$ at once. The constant $\delta = a/N$ does not depend on the point $x$. So (LB) does not just give continuity — it gives uniform continuity. We will only need the pointwise version below, but the stronger fact is yours for free.

4.Local boundedness implies the limit exists everywhere

Problem 3 Assume (LB). Then $f$ has a limit at every $x \in \R$, and $$\lim_{t \to x} f(t) = f(x).$$
Hint This is now almost no work. Fix $x \in \R$ and $\eps > 0$. Choose $N$ as in Problem 2 and put $\delta = a/N$. If $|t - x| < \delta$, what does Problem 2 immediately tell you about $|f(t) - f(x)|$?

5.The converse

The remaining problem is the converse. We show that if you happen to know that $f$ has a limit at every point, then in fact (LB) holds automatically. So the two conditions are equivalent.

Problem 4 Suppose $f$ has a limit at every $x \in \R$. Then there are $M > 0$ and $a > 0$ such that $$|f(x)| \le M \qquad \text{for all } x \in [-a, a].$$
Hint Only the limit at the single point $x = 0$ is needed. Call its value $L = \lim_{t \to 0} f(t)$. Take $\eps = 1$ in the definition of limit to produce a $\delta > 0$ such that $$0 < |t| < \delta \;\Longrightarrow\; |f(t) - L| < 1.$$ What does this give you for $|f(t)|$ when $0 < |t| < \delta$? Don't forget the point $t = 0$ itself — but recall $f(0) = 0$. Now choose $a$ slightly smaller than $\delta$ so that the closed interval $[-a, a]$ fits inside $(-\delta, \delta)$, and read off $M$.

What you have proved

Combining Problems 3 and 4:

Theorem.

Let $f:\R\to\R$ be additive. The following are equivalent.

  1. There exist $M > 0$ and $a > 0$ such that $|f(x)| \le M$ for every $x \in [-a, a]$.
  2. For every $x \in \R$, the limit $\displaystyle\lim_{t \to x} f(t)$ exists.

Moreover, when these hold, $\displaystyle\lim_{t \to x} f(t) = f(x)$ for every $x$, so $f$ is continuous on $\R$.

It is a beautiful fact, and not an obvious one, that boundedness on a single interval around the origin is enough to force continuity everywhere. The reason it works is the rigidity of the functional equation: an additive function is determined on all of $\R$ by what it does on any interval, however small.

Interlude — a glimpse of wildness

Before sketching whether wild additive functions actually exist on all of $\R$, here is a concrete object you can build by hand on a dense subset of $\R$. Let $$D \;=\; \Q + \Q\sqrt 2 \;=\; \{\, p + q\sqrt 2 : p, q \in \Q \,\}.$$ The set $D$ is countable, hence not all of $\R$, but it is dense in $\R$ since $\Q$ already is. Each element of $D$ has a unique representation as $p + q\sqrt 2$ with $p, q \in \Q$, because $\sqrt 2$ is irrational. So we may unambiguously define $f : D \to \R$ by $$f(p + q\sqrt 2) \;=\; p.$$

Problem 5 (optional). Show that $D$ is closed under addition and that $f$ is additive on $D$, that is, $f(\alpha + \beta) = f(\alpha) + f(\beta)$ for every $\alpha, \beta \in D$.
Hint Write $\alpha = p_1 + q_1\sqrt 2$ and $\beta = p_2 + q_2\sqrt 2$ in their unique form. Add them, collect like terms, and apply the definition of $f$ to all three of $\alpha$, $\beta$, $\alpha + \beta$. The single algebraic fact you need is that the representation $p + q\sqrt 2$ is unique, which forces the rational and $\sqrt 2$-coefficients to be added independently.

Two things to notice. First, $f$ cannot possibly be of the form $cx$: from $f(1) = 1$ we would need $c = 1$, but then $f(\sqrt 2) = \sqrt 2 \ne 0$. So $f$ is additive but not linear. Second, the graph of $f$ restricted to $D$ is already dense in $\R^2$; in particular, $f$ is unbounded on every interval around $0$, even just for inputs drawn from $D$.

But $f$ is so far only defined on the countable subset $D$. The values of any extension of $f$ to all of $\R$ are completely unconstrained by what we have done. So the question is sharp: can $f$ be extended to an additive function on all of $\R$? If yes, the extension is a wild additive function on $\R$, and the theorem of this project becomes substantive rather than vacuous.

Coda — additive functions that are not continuous

The answer to the interlude's question is yes, but the extension is delicate. It rests on a single deep fact about the real numbers, which we will not prove here. It depends on the axiom of choice, and no explicit instance of $B$ can be written down.

The basis property.

There exists an uncountable subset $B \subseteq \R$ with the following property: every real number $x$ admits a unique representation $$x \;=\; q_1 b_1 + q_2 b_2 + \cdots + q_n b_n$$ as a finite sum, where the $b_i \in B$ are distinct and the coefficients $q_i \in \Q$ are nonzero rationals.

Two comments are in order. First, although $B$ itself is uncountable, each individual $x$ uses only finitely many elements of $B$; different reals may use different finite subsets. Second, while $B$ cannot be exhibited explicitly, particular elements can be shown to lie in some such $B$. For example, the pair $\{1, \sqrt 2\}$ from the interlude is rationally independent — $\sqrt 2$ is not a rational multiple of $1$ — so it is consistent with the basis property and can be extended to some $B$.

The construction

Fix any such $B$, and choose a function $g: B \to \R$ however you like. There are no constraints whatsoever on $g$ — you simply pick a real value at each basis element. Now define $F : \R \to \R$ by $$F(q_1 b_1 + \cdots + q_n b_n) \;=\; q_1 g(b_1) + \cdots + q_n g(b_n).$$ The basis property guarantees that this formula assigns exactly one value to each $x \in \R$, so $F$ is a well-defined function. The verification that $F$ is additive is the same calculation as in the interlude, with the basis $\{1, \sqrt 2\}$ replaced by an arbitrary finite collection of basis elements.

Recovering the interlude's example

The function $f$ from the interlude is exactly what this construction produces if we take a basis $B$ that contains $1$ and $\sqrt 2$ and set $$g(1) = 1, \qquad g(\sqrt 2) = 0,$$ with arbitrary values on the rest of $B$. The defining formula then gives $F(p + q\sqrt 2) = p$ for all rationals $p, q$, agreeing with the interlude's $f$ on $D$. The construction has thereby extended the partial function $f$ from the dense subset $D$ to all of $\R$, while preserving additivity — which is exactly the question the interlude posed.

What the project's theorem then forces

The contrapositive of what you proved says: an additive function that is not continuous cannot satisfy (LB). So this $F$, being additive but not linear (hence not continuous), must be unbounded on every interval $[-a, a]$ around $0$, no matter how small $a$ is. The figure in the interlude already showed this unboundedness on the dense subset $D$; what the theorem now tells us is that the same wildness persists at the new points $\R \setminus D$ where the construction was free to do anything.

So the theorem of this project is tight: condition (LB) is exactly the dividing line between the tame additive functions $f(x) = cx$ and the wild ones produced by the construction above. Weakening (LB) is impossible — the wild functions are there, waiting to be missed — and ruling out the wild functions requires some condition like (LB).
