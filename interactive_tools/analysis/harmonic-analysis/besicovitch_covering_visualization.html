Besicovitch Covering Theorem: an interactive picture

The theorem says that from a bounded-radius family of balls centered at a set A, one can extract countably many disjoint subfamilies whose union still covers the centers. This page visualizes the greedy selection and the later coloring into disjoint layers.

The constants in the proof are deliberately not sharp. The visualization uses a finite two-dimensional model, so it illustrates the mechanism rather than the exact optimal value of the Besicovitch constant.

What the animation is showing

A \subseteq \bigcup_{i=1}^{N_n}\;\bigcup_{B\in\mathcal G_i} B, \qquad \text{each }\mathcal G_i\text{ disjoint.}
  1. Choose large available balls. At each step, pick a ball whose center is not yet covered and whose radius is close to maximal among the remaining possible centers.
  2. Cover all centers. The chosen balls cover the center set A. In the infinite bounded case, the selected radii must tend to zero.
  3. Bound overlaps. A packing argument bounds how many earlier selected balls can intersect a later selected ball.
  4. Color the intersection graph. If each selected ball intersects at most Mn−1 earlier selected balls, a greedy coloring separates the selected balls into at most Mn disjoint families.
  5. Unbounded case. Split space into thick annular shells and alternate odd/even shells, giving a finite doubling of the number of families.

How to use this page

Start with Step greedy choice. After several steps, press Show shrunk balls to see the packing idea: smaller concentric balls are disjoint. Then press Color into families to see the selected balls partitioned into layers of pairwise disjoint balls.

In the proof, the shrunk balls are used to bound the number of comparable-size balls meeting a fixed ball, while the angle estimate bounds the number of much larger balls meeting it.
