01 — The Idea

Taylor's theorem extends to functions of several variables: near a point (x₀, y₀), a smooth function f(x, y) can be approximated by a polynomial built from the function's partial derivatives evaluated at that point. The higher the order, the more closely the polynomial hugs the surface.

02 — Order 0: Constant

The simplest approximation is just the function value at the expansion point:

T₀(x, y) = f(x₀, y₀)

This gives a flat horizontal plane tangent to the surface only at the single point.

03 — Order 1: Tangent Plane

Adding first-order partial derivatives yields the tangent plane:

T₁ = f(x₀,y₀) + fₓ(x₀,y₀)(x−x₀) + f_y(x₀,y₀)(y−y₀)

This is the best linear approximation — it matches both the value and the slope in every direction at the point.

04 — Order 2: Osculating Paraboloid

Including second-order terms captures the curvature of the surface:

T₂ = T₁ + ½ fₓₓ(x−x₀)² + fₓᵧ(x−x₀)(y−y₀) + ½ f_yy(y−y₀)²

This quadratic surface matches the original function's value, gradient, and Hessian at (x₀, y₀). The error is O(‖Δ‖³).

05 — General Formula

For a function f : ℝⁿ → ℝ expanded around point a, the order-k Taylor polynomial is:

Tₖ(x) = Σ_{|α| ≤ k} (1/α!) · Dᵅf(a) · (x − a)ᵅ

where α = (α₁, …, αₙ) is a multi-index, |α| = α₁ + ⋯ + αₙ, α! = α₁!⋯αₙ!, and Dᵅ denotes the corresponding mixed partial derivative. The remainder satisfies |f(x) − Tₖ(x)| ≤ C ‖x − a‖ᵏ⁺¹ for x near a.
