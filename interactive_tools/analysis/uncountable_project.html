Analysis · Project I ℝ is not countable For the student
A guided project in four parts

The Uncountability
of the Real Line

You will discover — not be told — why no list can exhaust $[0,1]$.

In this project you will prove that the set of real numbers is not a countable set; such a set is called uncountable. The strategy: suppose, for contradiction, that some function $T : \mathbb{N} \to [0,1]$ is surjective. You will then build, step by step, a real number $A \in [0,1]$ that cannot equal $T(n)$ for any $n$ — a direct contradiction.

How to use this project

  • Each part opens with a Problem — a claim to prove.
  • Before peeking at anything, sit with the Think about prompts. Sketch on paper.
  • If you get stuck, expand the hints in order. Each reveals a little more.
  • Write your own proof first. Only then expand the Solution to compare.
  • End with the Reflect questions — they often matter more than the proof itself.

Warm-Up — A Preliminary Lemma

The Splitting Lemma

Before tackling the four parts of the project, prove this small but essential lemma. You will use it repeatedly.

Lemma (to prove) Let $[c,d]$ be a closed interval with $c < d$, and let $x \in \mathbb{R}$. Then there exists a closed subinterval $[c', d'] \subset [c, d]$ with $c' < d'$ and $x \notin [c', d']$.

Think about

  • Why does the condition $c' < d'$ matter? What would go wrong if we allowed $c' = d'$?
  • Where in $[c,d]$ could $x$ lie? Consider the cases $x \notin [c,d]$, $x = c$, $x = d$, and $c < x < d$ separately — or find a single argument that handles them all.
  • Do you really need to avoid $x$ "by a lot," or is any closed subinterval missing $x$ good enough?
Write out your own construction before opening any hints.
Hint 1A single picture handles every case.

Let $m = \tfrac{c+d}{2}$ be the midpoint. Any point $x \in \mathbb{R}$ satisfies either $x \leq m$ or $x > m$. In each case, one of the two halves $[c,m]$ or $[m,d]$ is "safe" from $x$ — and you can then shrink that safe half a bit more to be sure.

Hint 2Pick your subinterval inside the safe half.

If $x \leq m$, consider $\bigl[\tfrac{m+d}{2},\, d\bigr]$. Verify directly that $x \notin \bigl[\tfrac{m+d}{2},\, d\bigr]$.

If $x > m$, consider $\bigl[c,\, \tfrac{c+m}{2}\bigr]$. Verify directly that $x \notin \bigl[c,\, \tfrac{c+m}{2}\bigr]$.

SolutionProof of the Splitting Lemma.

Let $m = (c+d)/2$.

Case $x \leq m$. Set $[c', d'] = \bigl[\tfrac{m+d}{2}, d\bigr]$. Then $c' = \tfrac{m+d}{2} > m \geq x$, so $x < c' \leq d'$, giving $x \notin [c', d']$. Also $c < m < c' < d' = d$, so $[c', d'] \subset [c, d]$ and $c' < d'$.

Case $x > m$. Set $[c', d'] = \bigl[c, \tfrac{c+m}{2}\bigr]$. Then $d' = \tfrac{c+m}{2} < m < x$, so $x > d' \geq c'$, giving $x \notin [c', d']$. Also $c = c' < d' < m < d$, so $[c', d'] \subset [c, d]$ and $c' < d'$.

Reflect

Why did you need the interval $[c,d]$ to have positive length? Where in your proof would the argument fail if $c = d$?

Could you have proved the lemma using open intervals instead? Which step of the overall project makes closed intervals essential?

I
Part One — Construction

Building the Nested Tower

Problem 1 Show that there exist sequences $\{a_n\}_{n=1}^{\infty}$ and $\{b_n\}_{n=1}^{\infty}$ such that $[a_1, b_1] \subset [0,1]$, and for each $n \in \mathbb{N}$, $[a_{n+1}, b_{n+1}] \subset [a_n, b_n]$ and $T(n) \notin [a_n, b_n]$.

Think about

  • The sequences are infinite. What proof technique constructs something for every $n \in \mathbb{N}$?
  • You have already done the "one-step" work in the warm-up. How does the Splitting Lemma apply here?
  • What do you use as the ambient interval at step $n+1$? And what point must you avoid?
Draft your construction. What is the base case? What is the inductive step?
Hint 1This is induction in disguise.

You are asked to produce infinitely many intervals with certain properties. Use induction: construct $[a_1, b_1]$ explicitly (the base case), then explain how to get $[a_{n+1}, b_{n+1}]$ from $[a_n, b_n]$ (the inductive step).

Hint 2Apply the Splitting Lemma at each step.

Base case. Apply the Splitting Lemma to $[c,d] = [0,1]$ and the point $x = T(1)$. Let $[a_1, b_1]$ be the subinterval the lemma gives you.

Inductive step. Suppose $[a_n, b_n]$ has been defined with $a_n < b_n$. Now apply the Splitting Lemma to $[c,d] = [a_n, b_n]$ and the point $x = T(n+1)$.

Hint 3Check: does every condition hold?

Walk through the three required properties one by one:

(i) $[a_1, b_1] \subset [0,1]$? — This is in the conclusion of the Splitting Lemma applied in the base case.

(ii) $[a_{n+1}, b_{n+1}] \subset [a_n, b_n]$? — This is in the conclusion of the Splitting Lemma applied in the inductive step.

(iii) $T(n) \notin [a_n, b_n]$? — This is exactly how you chose each interval to dodge the respective $T(n)$.

SolutionInductive construction of the nested tower.

Base case. By the Splitting Lemma applied to $[0,1]$ and $T(1)$, there exists a closed interval $[a_1, b_1] \subset [0,1]$ with $a_1 < b_1$ and $T(1) \notin [a_1, b_1]$.

Inductive step. Assume $[a_n, b_n]$ has been defined with $a_n < b_n$. By the Splitting Lemma applied to $[a_n, b_n]$ and $T(n+1)$, there exists a closed interval $[a_{n+1}, b_{n+1}] \subset [a_n, b_n]$ with $a_{n+1} < b_{n+1}$ and $T(n+1) \notin [a_{n+1}, b_{n+1}]$.

By induction the sequences $\{a_n\}, \{b_n\}$ are defined for every $n \in \mathbb{N}$ and satisfy the three required conditions.

Reflect

Your construction works for any function $T : \mathbb{N} \to [0,1]$. Notice that the sequences $\{a_n\}, \{b_n\}$ depend on $T$ — different $T$'s give different towers. Why is that fine?

Could the construction be made to yield $b_n - a_n \to 0$? (You won't need this, but it's worth asking.)

II
Part Two — Convergence

The Sequence $\{a_n\}$ Has a Limit

Problem 2 Show that $\{a_n\}_{n=1}^{\infty}$ converges. Call the limit $A$.

Think about

  • The containment $[a_{n+1}, b_{n+1}] \subset [a_n, b_n]$ tells you two inequalities. Which one is about the $a_n$'s?
  • Which convergence theorems do you know that apply to sequences of real numbers without needing to guess the limit in advance?
  • Is $\{a_n\}$ bounded? By what? (Hint: you don't need to know $A$ to bound $a_n$.)
Identify the two properties of $\{a_n\}$ you need, then name the theorem that closes the argument.
Hint 1Extract monotonicity from containment.

If $[a_{n+1}, b_{n+1}] \subset [a_n, b_n]$, then every point of $[a_{n+1}, b_{n+1}]$ lies in $[a_n, b_n]$. In particular, the left endpoint $a_{n+1}$ lies in $[a_n, b_n]$, so $a_n \leq a_{n+1}$.

Hint 2Extract a bound.

By the same containment reasoning, $b_{n+1} \leq b_n$. Iterating gives $b_n \leq b_1$, and since every $a_n \leq b_n$, we get $a_n \leq b_1 \leq 1$.

Hint 3Name the theorem.

You now have a sequence that is monotonically non-decreasing and bounded above. The Monotone Convergence Theorem guarantees that such a sequence converges in $\mathbb{R}$.

SolutionWhy $\{a_n\}$ converges.

Since $[a_{n+1}, b_{n+1}] \subset [a_n, b_n]$, the point $a_{n+1}$ lies in $[a_n, b_n]$, forcing $a_n \leq a_{n+1}$. Thus $\{a_n\}$ is monotonically non-decreasing.

Similarly $b_{n+1} \leq b_n$, so by induction $b_n \leq b_1$. Combined with $a_n \leq b_n$, this gives $a_n \leq b_1 \leq 1$ for every $n \in \mathbb{N}$, so $\{a_n\}$ is bounded above.

By the Monotone Convergence Theorem, $\{a_n\}$ converges. Define $$A := \lim_{n \to \infty} a_n = \sup_{n \in \mathbb{N}} a_n. \qquad \blacksquare$$

Reflect

The Monotone Convergence Theorem is a form of the completeness of $\mathbb{R}$. Where exactly does completeness enter? What would fail if we tried to run this whole argument inside $\mathbb{Q}$?

Does $\{b_n\}$ also converge? To the same limit as $\{a_n\}$, or possibly a different one?

III
Part Three — Entrapment

$A$ Lies in Every Interval — and Outside $\operatorname{im} T$

Problem 3 Prove that $A \in [a_n, b_n]$ for each $n \in \mathbb{N}$. Conclude that $A \notin \operatorname{im} T$.

Think about

  • You want to show $a_n \leq A \leq b_n$. What do you know about $A$? (You don't know its exact value — only that it is a certain limit.)
  • For a fixed $n$, what happens to $a_m$ when $m \geq n$? And how do weak inequalities behave under limits?
  • Once you have $A \in [a_n, b_n]$ for every $n$, how do you extract "$A \neq T(n)$ for every $n$" from it?
Fix an arbitrary $n$. Try to squeeze $A$ between $a_n$ and $b_n$ using tails of the sequence.
Hint 1Compare a tail of $\{a_m\}$ to $[a_n, b_n]$.

Fix $n$. For every $m \geq n$, the iterated containment gives $[a_m, b_m] \subset [a_n, b_n]$. In particular, $$a_n \leq a_m \leq b_m \leq b_n \qquad \text{for all } m \geq n.$$

Hint 2Take $m \to \infty$.

The sequence $\{a_m\}_{m \geq n}$ is a tail of $\{a_m\}$, so it converges to the same limit $A$. Limits preserve weak inequalities: $$a_n \leq a_m \leq b_n \quad \text{for all } m \geq n \quad \Longrightarrow \quad a_n \leq A \leq b_n.$$

Hint 3Close the argument by contradiction.

Suppose $A = T(k)$ for some $k \in \mathbb{N}$. By Part 1, $T(k) \notin [a_k, b_k]$. But you just showed $A \in [a_k, b_k]$. These two facts cannot both hold for the same point.

Solution$A$ is in every $[a_n, b_n]$ and hence outside $\operatorname{im} T$.

Step 1: $A \in [a_n, b_n]$. Fix $n \in \mathbb{N}$. For every $m \geq n$, iterated containment yields $[a_m, b_m] \subset [a_n, b_n]$, so $$a_n \leq a_m \leq b_m \leq b_n.$$ Since $\{a_m\}_{m \geq n}$ is a tail of the convergent sequence $\{a_m\}$, it converges to $A$. Passing to the limit in the weak inequalities above: $$a_n \leq A \leq b_n,$$ i.e., $A \in [a_n, b_n]$.

Step 2: $A \notin \operatorname{im} T$. Suppose for contradiction that $A = T(k)$ for some $k \in \mathbb{N}$. By Part 1, $T(k) \notin [a_k, b_k]$, while by Step 1, $A \in [a_k, b_k]$. This contradicts $A = T(k)$. Hence $A \neq T(n)$ for every $n \in \mathbb{N}$, i.e., $A \notin \operatorname{im} T$.

Reflect

The proof used "limits preserve weak inequalities" but not strict ones. If we had $a_m < b_n$ for all $m$, we could not conclude $A < b_n$ — only $A \leq b_n$. Where would a strict inequality have been false?

The argument shows something stronger than asked: it exhibits a specific $A$ missing from the image of $T$. In what sense is this "constructive"?

IV
Part Four — Conclusion

Finishing the Proof that $[0,1]$ Is Uncountable

Problem 4 Use Parts 1–3 to finish the proof that $[0,1]$ is uncountable. Then extend the conclusion to $\mathbb{R}$.

Think about

  • What is the precise definition of "countable"? What would it mean for $[0,1]$ to be countable?
  • Parts 1–3 began "let $T : \mathbb{N} \to [0,1]$ be any function." That "any" is doing a lot of work. What does it let you conclude?
  • Why does uncountability of $[0,1]$ imply uncountability of $\mathbb{R}$? What general principle about countability and subsets are you using?
Structure the conclusion as a proof by contradiction. What do you assume, and what contradicts what?
Hint 1Pin down "countable."

A set $S$ is countable iff there exists a surjection $T : \mathbb{N} \to S$. Equivalently, $S$ can be written as the image of some sequence. So $[0,1]$ is countable iff some $T : \mathbb{N} \to [0,1]$ has $\operatorname{im} T = [0,1]$.

Hint 2Set up the contradiction.

Suppose $[0,1]$ were countable. Then some $T : \mathbb{N} \to [0,1]$ is surjective, i.e., $\operatorname{im} T = [0,1]$. Now apply Parts 1–3 to this $T$. What do you get? Where's the contradiction?

Hint 3Extend to $\mathbb{R}$.

Use the following fact: every subset of a countable set is countable. Contrapose it: if $S$ contains an uncountable subset, then $S$ is uncountable. Now apply this to $\mathbb{R} \supset [0,1]$.

SolutionConclusion of the proof.

$[0,1]$ is uncountable. Suppose, for contradiction, $[0,1]$ were countable. Then there is a surjection $T : \mathbb{N} \to [0,1]$, so $\operatorname{im} T = [0,1]$. By Parts 1–3 applied to this $T$, there exists $A$ with $$A \in [a_1, b_1] \subset [0,1] \qquad \text{and} \qquad A \notin \operatorname{im} T.$$ Thus $A \in [0,1] = \operatorname{im} T$ and $A \notin \operatorname{im} T$ — contradiction. Therefore no surjection $T : \mathbb{N} \to [0,1]$ exists, and $[0,1]$ is uncountable.

$\mathbb{R}$ is uncountable. Every subset of a countable set is countable (restrict a surjection from $\mathbb{N}$). Contrapositively, any set containing an uncountable subset is uncountable. Since $[0,1] \subset \mathbb{R}$ and $[0,1]$ is uncountable, $\mathbb{R}$ is uncountable.

Reflect

The proof never used a single specific property of the function $T$ — no formula, no decimal expansion, no pattern. It worked for every $T : \mathbb{N} \to [0,1]$. What makes this level of generality possible?

Cantor's diagonal argument is the more famous proof of this theorem. Can you sketch it and identify the analogue of our point $A$?

Which of the following sets are countable, and which are uncountable? Try each before deciding. $\mathbb{Q}$, $\mathbb{Q} \cap [0,1]$, $\mathbb{R} \setminus \mathbb{Q}$, the set of algebraic numbers, the Cantor set, the set of all finite sequences of rationals, the power set $\mathcal{P}(\mathbb{N})$.
